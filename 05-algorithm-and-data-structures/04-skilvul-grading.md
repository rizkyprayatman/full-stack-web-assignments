# Program Skilvul Grading

## Declaration

```
var nilaiAwal = number;
var nilaiAkhir = number;
```

## Implementation

```
START

INPUT nilaiAwal

if nilaiAwal > 70 then
    DISPLAY nilaiAwal
else if (nilaiAwal + 1) % 5 = 0 then
    nilaiAkhir = nilaiAwal + 1
    DISPLAY nilaiAkhir
else if (nilaiAwal + 1) % 5 = 0 then
    nilaiAkhir = nilaiAwal + 2
    DISPLAY nilaiAkhir
end if

END
```
