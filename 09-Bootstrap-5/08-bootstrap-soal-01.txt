Soal - 01

1. Apa itu CSS Framework ?
2. Apakah bootstrap satu-satunya CSS Framework yang dapat kita gunakan ?
3. Sebutkan beberapa CSS Framework selain Bootstrap
4. Bagaimana tata cara penggunaan Bootstrap ?


Jawaban - 01

1. CSS Framework ialah library CSS yang digunakan untuk standarisasi Layout,
   serta mempeercepat dalam proses development.
2. bukan satu-satunya, karena ada banyak pilihan CSS Framework selain Bootstrap.
3. Berikut beberapa CSS Framework selain Bootstrap
   - Materialize CSS
   - Tailwind
   - Foundation
   - Semantic UI
   - Miligram
4. Penggunaan Bootstrap dapat dilakukan dengan beberapa cara yakni
   - Menginstall menggunakan npm
   - Copy-paste stylesheet <link> ke <head> (Dalam development mengharuskan Online), atau
   - Download dan meletakkan bootstrap dalam project kemudian menautkan filenya 
     (development dalam mode offline dapat dilakukan)
