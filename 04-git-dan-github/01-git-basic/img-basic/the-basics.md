# Git Basic

Berikut langkah yang dilakukan

1. mkdir git-basic
2. cd git-basic
3. git init .
4. touch first.txt
5. git add .
6. git commit -m "adding first.txt"
7. git log
8. touch second.txt
9. git add .
10. git commit -m "adding first.txt"
11. git rm first.txt
12. git add .
13. git commit -m "removing first.txt"
14. git log
